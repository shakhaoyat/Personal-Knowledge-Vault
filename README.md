# Personal Knowledge Vault

A static landing page for a personal knowledge management product. The site is built with plain HTML and CSS and uses assets from the `assets/` folder for the hero, feature cards, solution section, and footer branding.

## What It Includes

- Responsive-style landing page layout with a navigation bar, hero banner, feature cards, solution section, call-to-action area, and footer
- Google Fonts integration for typography
- Font Awesome icons for social links
- Section mockups and reference visuals in the `Sections/` folder

## Project Structure

- `index.html` - Main page markup
- `styles/styles.css` - Site styling
- `assets/` - Images and UI assets used by the page
- `Sections/` - Reference screenshots for each section
- `UI/` - Supporting design files

## How To Run

1. Open `index.html` in a browser.
2. Or use a local server extension such as Live Server in VS Code for a better preview workflow.

## Notes

This project does not use a build step or JavaScript framework. It is intended as a simple front-end implementation of the provided design.
